# BlockedIps
## Описание
**BlockedIps** – это проект, предназначенный для создания и управления черным списком IP-адресов,
 подозреваемых в слежке в интернете. С помощью этого инструмента вы сможете защищать свою сеть,
  блокируя нежелательных пользователей и предотвращая атаки на вашу систему.
  
  ---
  
## Description
**BlockedIps** is a project designed to create and manage a blacklist of IP addresses
suspected of internet stalking. This tool allows you to protect your network by blocking
unwanted users and preventing attacks on your system.
